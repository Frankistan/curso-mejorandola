Curso de Javascript para aplicaciones web

Viernes 3 y sábado 4 de agosto / México DF

Javascript es el lenguaje central del desarrollo web. Creando la interacción con el usuario en el lado del navegador y, gracias a Node.js, también del lado del servidor.

JS es mucho más. jQuery, Backbone.js y Phonegap crean un ecosistema de complejas apps móviles, en tablets y desktops.

Aprende las tecnologías que convirtieron a Javascript en un lenguaje de primera para robustas y complejas experiencias en internet.
