Empecemos con la sintaxis de javascript

La sintaxis de javascript es parecida a la de C, con un poco de contribuciones de java, scheme y smalltalk

Variables

                    var foo = 'variable';

Condicionales(if)

                    if(foo === "variable"){
                        console.log(true);
                    }else{
                        console.log(false);
                    }

Tipos de datos en javascript

Boleano

var Boleano = true || false;

Numérico

var Numerico = 4;

Texto(string)

var Texto = "(string)";

Objetos

var foo = {
    "key"      : "data",
    "otherKey" : "moreData"
}

Funciones

                
var miFuncion = function(algunos, parametros){
    algunos    = algunos || "algunos";
    parametros = parametros || "parametros";
    // hacemos unas cuantas cosas ..

    console.log(algunos, parametros);
}
