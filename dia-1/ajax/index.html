Ajax

Un poco de historia

En el 2000 Microsoft agrega el objeto XMLHttpRequest object al browser y para el 2004 Google lo usa en Gmail y es el punto en el que el mundo lo empieza a usar.

Â¿Que ha cambiado?

Bastante, a mejorado su seguridad, dejado de usar el xml, agregado sockets para substituir "ajax".

En jQuery 1.5 cambiaron el modulo de ajax para correr sobre promesas.

var promesa = $.ajax('/dia-1/ajax/data.json');

promesa.done(function(data){
    console.log('informacion lista', data);
});

console.log('Procesar mas cosas mientras llega nuestra informacion');

Un ejemplo un poco mas complicado

var template = $.ajax('/dia-1/ajax/template.json');
var list     = $.ajax('/dia-1/ajax/list.json');
$.when(template, list).then(function(template, data){
    var templateData = template[0];
    var listData     = data[0];
    
    $.template( "list", templateData.template );
    $.tmpl( "list", listData ).appendTo('.full_contenidos'); 
});
