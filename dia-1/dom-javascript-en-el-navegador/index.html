Templates para agregar muchos nodos de golpe

Los templates de jquery funcionan en 3 sencillos pasos

  1. Conseguir el template 
     var listTemplate = $('#list-template').html();
  2. Compilar el template 
     $.template( "list", listTemplate );
  3. Agregar el informacion al template 
     $.tmpl( "list", window.listData );
