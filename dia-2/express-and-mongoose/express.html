Express

Express es un DSL para crear aplicaciones web de manera rapida

SinatraExpress is a DSL for quickly creating web applications in RubyNode with minimal effort.

Para crear una aplicacion con Express de necesita crear un objeto aplicacion

var express = require('express');

var app = express();

Agregar los plugins necesarios

app.use(express.static('./public'));

//View Engine
app.engine('html', cons.jqtpl);
// set .html as the default extension 
app.set('view engine', 'html');
app.set('views', './views');

Agregar las rutas con las que trabajaremos

app.get('/', function(req, res){
  	res.render('index',{ pageName : 'Home' });
});

app.get('/to-do/:name',function (req, res){
	res.render('index',{ pageName : req.params.name})
});

Decirle a la aplicacion que escuche en algun puerto

app.listen(3000);
