Mongoose

Mongoose es un ORM entre node y mongodb

Mongodb es una base de dato schemaless, pero Mongoose agregara schemas

Paso 1 crear nuestros Schemas

var ToDo = new Schema({
    id      : ObjectId,
    title     : String,
    completed : Boolean,
});

var ToDoList = new Schema({
    id      : ObjectId,
    title     : String,
    toDos     : [ToDo]  
});

Crear nuestros modelos

var List = mongoose.model('ToDoList', ToDoList);

List.prototype.getToDosTitles = function() {
  return this.toDos.map(function(toDo){
    return toDo.title;
  });
};

Podemos hacer busquedas con nuestros modelos o crear nuevas instancias
