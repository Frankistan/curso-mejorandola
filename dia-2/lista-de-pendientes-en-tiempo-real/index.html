Lista de pendientes en tiempo real

Primer paso

Instalemos los paquetes necesarios para este proyecto

npm install .
npm install nodemon -g

Paso 2

Necesitamos juntar express con mongoose

Tomemos el server del ejercicio de express y agregemos los schemas y el modelo de mongoose

Despues en el llamada al router '/to-do/:name' entregemos la lista de mongoose

Paso 3

Agregemos el estilo y la logica para renderearlo como lista de tareas

Esto implica traer todo el ejercicio de ayer a la vista y archivos estaticos de este proyecto

Paso 4

Notificar al server cuando se modifica la lista y que informacion a cambiado de ella

Paso 5

Paso subscribirnos al server para escuchar cuando alguien mas cambia la lista
